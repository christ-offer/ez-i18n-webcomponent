Welcome

This is a relatievely simple i18n web-component - it works for me.

Basic Usage:

Pretty straight forward markup:

<i18n-selector hidden>
<select>
  <option value="en">English</option>
  <option value="es">Spanish</option>
  <option value="fr">French</option>
</select>
</i18n-selector>
<h1 data-i18n="welcome">Welcome</h1>

// Initialize the component with translations
document.addEventListener("DOMContentLoaded", () => {
  const i18n = document.querySelector("i18n-selector");
  i18n.addLanguages(translations);

  // Example of listening to language changes
  i18n.addEventListener("languageChanged", (e) => {
    console.log(`Language changed to: ${e.detail.language}`);
  });
});

If you have to have nested translations (e.g. styling a single word/phrase in a paragraph):

How to do it:

<p>
  <span data-i18n="intro.1">This is a sample text that</span>
  <span data-i18n="intro.span">definitely</span>
  <span data-i18n="intro.2"> will be translated.</span>
</p>

This is a sample text that definitely will be translated.

How Not to do it:

<p data-i18n="intro.text">
  This is a sample text that
  <span data-i18n="intro.span">definitely</span>
  translates - but where did 'definitely' go?
</p>

This is a sample text that definitely translates - but where did 'definitely' go?
